SalesHistory Agent

This policy explains what data the extension accesses, how it is used, and where it is sent.

Overview

SalesHistory Agent is a Chrome extension that looks up publicly available sales history for trading cards on eBay, Fanatics Collect, and COMC.

The extension does not collect, store, or transmit any personal data, browsing history, or analytics.

Data Processed

The only data the extension processes is:

  • Card listing titles — extracted from the page you are viewing to build search queries.
  • User preferences — stored locally in your browser via chrome.storage.sync (trigger key, toggle settings).
  • Gemini API key (optional) — if you enable AI title cleaning, your API key is stored locally in chrome.storage.sync and is only sent to Google's Gemini API endpoint.
External Services

When you trigger a sales lookup, the extension sends the card title as a search query to:

  • E
    eBay www.ebay.com — retrieves sold listing results
  • 1
    130point.com back.130point.com — retrieves historical sales data
  • G
    Google Gemini API generativelanguage.googleapis.com — cleans card titles for better searches (only if AI cleaning is enabled)

All requests are made directly from your browser. The extension does not operate or communicate with any server controlled by the developer.

Data Storage

User preferences and the optional Gemini API key are stored in chrome.storage.sync, which syncs across your signed-in Chrome instances. No data is stored on external servers.

Third-Party Access

The extension does not share, sell, or transfer your data to any third party beyond the API requests described above, which are necessary for the extension to function.

Permissions
  • Storage — to save your preferences locally.
  • Host permissions — to fetch sales data from eBay, 130point.com, and the Gemini API.
  • Content script access — to read card titles on supported sites (eBay, Fanatics Collect, COMC, Basket Place) and display the overlay.
Changes to This Policy

If this policy is updated, the changes will be posted here with a revised date. Continued use of the extension after changes constitutes acceptance.
